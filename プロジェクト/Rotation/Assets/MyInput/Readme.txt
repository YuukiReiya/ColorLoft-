///////////////////////////////////////////////////////////////////////////////////////////////////
//											使い方												 //
///////////////////////////////////////////////////////////////////////////////////////////////////
・フォルダのprefabから"MyInputManager"をヒエラルキーにドラッグアンドドロップ
・インスペクタから値を変更して"Seeting on GamePad"をクリックしてください
(または、MyInputManagerスクリプトを任意のオブジェクトに付けることでも設定が可能です。)


※InputManagerの値を直接書き換えてしまうので個々で設定していたものが消えてしまうことに注意してください
